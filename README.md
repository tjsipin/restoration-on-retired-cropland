# restoration-on-retired-cropland
This repository provides the code used to clean, analyze, and plot the data in "Climate-resilient native plant restoration on retired croplands in California" (submitted to Agriculture, Ecosystems & Environment).

Authors:
TJ Sipin, Nickolas McManus, Madison Calbert, William Dean, Ellen Cypher, Amy Quandt, Ashley Larsen


Data can be found here: https://doi.org/10.5281/zenodo.19458425
